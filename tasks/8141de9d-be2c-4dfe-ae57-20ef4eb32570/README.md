# Task Workspace

This directory contains the live outputs for one orchestration run.

## Layout

- `input/`: original requirement and run request metadata
- `cycles/cycle-xx/requirements`: PC outputs
- `cycles/cycle-xx/architecture`: CA outputs
- `cycles/cycle-xx/workspace/frontend`: FD generated frontend code
- `cycles/cycle-xx/workspace/backend`: BD generated backend code
- `cycles/cycle-xx/delivery`: DE delivery bundle
- `cycles/cycle-xx/quality`: QT reports and remediation notes
