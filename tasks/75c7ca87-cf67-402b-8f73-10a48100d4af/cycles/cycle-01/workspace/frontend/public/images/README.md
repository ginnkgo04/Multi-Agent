# 图片资源目录

此目录包含北极熊保护网站所需的所有图片资源。

## 目录结构

```
images/
├── polar-bear-hero.jpg          # 首页英雄区域背景图片
├── polar-bear-threats.jpg       # 威胁展示区域图片
├── polar-bear-habitat.jpg       # 栖息地展示图片
├── polar-bear-family.jpg        # 北极熊家庭图片
├── conservation-actions.jpg     # 保护行动展示图片
├── organization-logo.png        # 组织Logo
├── favicon.ico                  # 网站图标
└── social-share.jpg             # 社交媒体分享图片
```

## 图片规格要求

### 1. 首页英雄图片 (polar-bear-hero.jpg)
- 尺寸：1920×1080 像素
- 格式：JPEG，优化压缩
- 内容：北极熊在冰雪环境中的高质量照片
- 文件大小：< 300KB

### 2. 威胁展示图片 (polar-bear-threats.jpg)
- 尺寸：1200×800 像素
- 格式：JPEG
- 内容：展示气候变化对北极熊栖息地的影响
- 文件大小：< 200KB

### 3. 栖息地图片 (polar-bear-habitat.jpg)
- 尺寸：1200×800 像素
- 格式：JPEG
- 内容：北极熊的自然栖息地环境
- 文件大小：< 200KB

### 4. 北极熊家庭图片 (polar-bear-family.jpg)
- 尺寸：1000×800 像素
- 格式：JPEG
- 内容：母熊和幼崽的温馨场景
- 文件大小：< 150KB

### 5. 保护行动图片 (conservation-actions.jpg)
- 尺寸：1200×600 像素
- 格式：JPEG
- 内容：保护工作者或志愿者活动场景
- 文件大小：< 180KB

### 6. 组织Logo (organization-logo.png)
- 尺寸：200×200 像素（正方形）
- 格式：PNG，透明背景
- 内容：北极熊保护组织Logo
- 文件大小：< 50KB

### 7. 网站图标 (favicon.ico)
- 尺寸：16×16, 32×32, 48×48 像素
- 格式：ICO
- 内容：简化版北极熊图标
- 文件大小：< 10KB

### 8. 社交媒体分享图片 (social-share.jpg)
- 尺寸：1200×630 像素（Facebook推荐尺寸）
- 格式：JPEG
- 内容：吸引人的北极熊图片和网站标题
- 文件大小：< 100KB

## 图片优化指南

### 1. 压缩要求
- 所有JPEG图片使用85%质量压缩
- PNG图片使用PNG-8格式（如适用）
- 使用工具如TinyPNG、ImageOptim进行优化

### 2. 响应式图片考虑
- 为不同屏幕尺寸准备多个版本
- 使用`srcset`属性提供不同分辨率图片
- 考虑使用WebP格式以获得更好压缩

### 3. 可访问性
- 所有图片必须包含有意义的alt文本
- 装饰性图片使用空alt属性（alt=""）
- 信息性图片提供详细描述

### 4. 版权注意事项
- 确保所有图片拥有合法使用权
- 优先使用原创或授权图片
- 如使用第三方图片，确保符合许可协议
- 考虑使用免费图库资源：
  - Unsplash
  - Pexels
  - Pixabay

## 图片命名规范

### 命名规则
- 使用小写字母
- 使用连字符分隔单词
- 描述性命名，避免通用名称
- 包含尺寸或用途信息

### 示例
```
polar-bear-hero-1920x1080.jpg
threat-climate-change-800x600.jpg
logo-organization-200x200.png
```

## 开发注意事项

### 1. 占位图片
在开发阶段，可以使用以下占位服务：
- https://placeholder.com
- https://picsum.photos
- https://via.placeholder.com

### 2. 懒加载
- 对非首屏图片实现懒加载
- 使用`loading="lazy"`属性
- 考虑使用Intersection Observer API

### 3. 图片CDN
生产环境建议使用CDN服务：
- Cloudinary
- Imgix
- AWS CloudFront

## 维护指南

### 1. 定期检查
- 每月检查图片加载性能
- 更新过时的图片内容
- 优化文件大小

### 2. 备份策略
- 保留原始高质量图片
- 定期备份图片资源
- 使用版本控制系统管理

### 3. 监控
- 监控图片加载错误
- 跟踪图片使用情况
- 分析用户与图片的互动

## 替代方案

如果无法获取实际北极熊图片，可以考虑：

### 1. 插画风格
- 使用北极熊主题插画
- 创建信息图表
- 使用图标系统

### 2. 数据可视化
- 气候变化数据图表
- 北极熊数量趋势图
- 保护成果可视化

### 3. 用户生成内容
- 鼓励用户分享保护故事
- 展示志愿者活动照片
- 创建图片墙展示支持者

## 紧急情况

如果图片资源暂时不可用，网站将：
1. 显示友好的占位图片
2. 保持文字内容完整可读
3. 提供图片加载失败提示
4. 允许用户手动刷新图片
