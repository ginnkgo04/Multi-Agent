# 沙县小吃网站前端实现说明

## 技术栈选择

### 核心框架
- **Next.js 14**：React框架，提供服务器端渲染、静态生成和优秀的开发体验
- **TypeScript**：类型安全，提高代码质量和开发效率
- **Tailwind CSS**：实用优先的CSS框架，快速构建响应式设计

### 图标库
- **Lucide React**：轻量级图标库，提供简洁美观的图标

## 设计系统

### 色彩方案
- **主色调**：琥珀色 (#d97706) - 代表美食、温暖和传统
- **辅助色**：灰色调 - 提供简洁、专业的背景
- **强调色**：绿色、蓝色、紫色 - 用于特色功能区分

### 排版系统
- **字体**：系统字体栈，确保跨平台一致性
- **字号层次**：
  - 标题：3xl-6xl (48-72px)
  - 副标题：xl-2xl (20-24px)
  - 正文：base (16px)
  - 小字：sm (14px)

### 间距系统
- 使用Tailwind的间距比例：4px基数
- 主要间距：4, 8, 12, 16, 20, 24, 32, 40, 48, 64

## 响应式设计

### 断点设置
- **移动端**：< 768px
- **平板**：768px - 1024px
- **桌面端**：> 1024px

### 响应式策略
1. **移动优先**：基础样式针对移动端，逐步增强
2. **弹性布局**：使用flex和grid实现自适应
3. **断点覆盖**：使用Tailwind的响应式前缀

## 开发说明

### 项目结构
```
workspace/frontend/
├── app/
│   ├── page.tsx          # 主页面
│   ├── globals.css       # 全局样式
│   └── layout.tsx        # 布局组件（待创建）
├── components/
│   └── FeatureExperience.tsx  # 特色体验组件
├── lib/
│   └── api.ts            # API工具函数
└── package.json          # 项目配置
```

### 运行项目
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 环境要求
- Node.js 18+ 
- npm 9+ 或 yarn 1.22+
- 现代浏览器支持
