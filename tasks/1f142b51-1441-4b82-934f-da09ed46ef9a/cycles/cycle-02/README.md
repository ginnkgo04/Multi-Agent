# 小熊猫介绍网站

一个介绍小熊猫（Ailurus fulgens）的单页网站，包含基本信息、栖息地、饮食、保护状态等内容，采用响应式设计。

## 项目结构

```
red-panda-website/
├── index.html              # 主HTML文件
├── style.css              # 样式文件
├── images/                # 图片资源目录
│   ├── red-panda-1.jpg
│   ├── red-panda-2.jpg
│   └── red-panda-3.jpg
├── README.md              # 项目说明文档
└── delivery/              # 交付文档
    ├── integration_report.md
    ├── run_manifest.json
    └── checklist.md
```

## 功能特性

- **响应式设计**：适配桌面、平板和手机设备
- **语义化HTML**：使用HTML5语义标签，提升可访问性
- **视觉吸引力**：精心设计的布局和配色方案
- **丰富内容**：包含小熊猫的全面介绍信息
- **无障碍访问**：所有图片都有alt文本描述

## 内容章节

1. **首页** - 小熊猫的简要介绍和主图展示
2. **基本信息** - 分类学、体型、寿命等基本信息
3. **栖息地** - 分布区域和栖息环境
4. **饮食习性** - 主要食物和觅食行为
5. **保护状态** - 濒危等级和保护措施
6. **趣味知识** - 有趣的小熊猫冷知识

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 媒体查询)
- 响应式图片
- 无JavaScript依赖

## 快速开始

1. 克隆或下载项目文件
2. 确保所有文件保持原有目录结构
3. 在浏览器中直接打开 `index.html` 文件

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 10+
- Edge 79+

## 开发说明

- 采用移动优先的响应式设计策略
- 使用CSS自定义属性（变量）管理颜色方案
- 所有图片都包含适当的alt文本，确保可访问性
- 代码结构清晰，便于维护和扩展

## 图片使用说明

项目中使用的图片均为示例图片，实际部署时建议：
1. 使用高质量、无版权或已获授权的小熊猫图片
2. 确保图片文件大小优化，以提高加载速度
3. 为所有图片提供描述性的alt文本
